# ATM Simulator System (Python GUI)

# Project Description
This project is a simple ATM Simulator System developed using Python Tkinter GUI.

## Features
- PIN Login Authentication
- Check Balance
- Deposit Money
- Withdraw Money
- Exit System

## Technologies Used
- Python
- Tkinter (GUI Library)

## Default Login
PIN: 1234  
Starting Balance: Rs. 1000

## How to Run
1. Install Python
2. Run:
   python atm_gui.py
